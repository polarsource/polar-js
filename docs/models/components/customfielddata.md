# CustomFieldData


## Supported Types

### `string`

```typescript
const value: string = "<value>";
```

### `number`

```typescript
const value: number = 840831;
```

### `boolean`

```typescript
const value: boolean = false;
```

### `Date`

```typescript
const value: Date = new Date("2024-06-28T23:10:28.798Z");
```
