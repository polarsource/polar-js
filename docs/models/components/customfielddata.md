# CustomFieldData


## Supported Types

### `string`

```typescript
const value: string = "<value>";
```

### `number`

```typescript
const value: number = 776750;
```

### `boolean`

```typescript
const value: boolean = false;
```

### `Date`

```typescript
const value: Date = new Date("2025-11-25T20:06:47.886Z");
```
