# CustomFieldData


## Supported Types

### `string`

```typescript
const value: string = "<value>";
```

### `number`

```typescript
const value: number = 954406;
```

### `boolean`

```typescript
const value: boolean = false;
```

### `Date`

```typescript
const value: Date = new Date("2023-01-24T01:52:59.582Z");
```
