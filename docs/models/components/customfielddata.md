# CustomFieldData


## Supported Types

### `string`

```typescript
const value: string = "<value>";
```

### `number`

```typescript
const value: number = 941743;
```

### `boolean`

```typescript
const value: boolean = false;
```

### `Date`

```typescript
const value: Date = new Date("2025-03-01T06:19:32.936Z");
```
