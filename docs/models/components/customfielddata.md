# CustomFieldData


## Supported Types

### `string`

```typescript
const value: string = "<value>";
```

### `number`

```typescript
const value: number = 926266;
```

### `boolean`

```typescript
const value: boolean = false;
```

### `Date`

```typescript
const value: Date = new Date("2024-12-26T07:01:46.533Z");
```
