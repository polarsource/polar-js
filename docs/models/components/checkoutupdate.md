# CheckoutUpdate

Update an existing checkout session using an access token.

## Example Usage

```typescript
import { CheckoutUpdate } from "@polar-sh/sdk/models/components/checkoutupdate.js";

let value: CheckoutUpdate = {
  customerName: "John Doe",
  customerBillingAddress: {
    country: "US",
  },
  locale: "en",
};
```
