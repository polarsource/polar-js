# CheckoutPriceCreate

Create a new checkout session from a product price.

**Deprecated**: Use `CheckoutProductsCreate` instead.

Metadata set on the checkout will be copied
to the resulting order and/or subscription.

## Example Usage

```typescript
import { CheckoutPriceCreate } from "@polar-sh/sdk/models/components/checkoutpricecreate.js";

let value: CheckoutPriceCreate = {
  productPriceId: "<value>",
};
```
