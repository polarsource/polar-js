# MetadataQuery


## Supported Types

### `string`

```typescript
const value: string = "<value>";
```

### `number`

```typescript
const value: number = 128403;
```

### `boolean`

```typescript
const value: boolean = true;
```

### `string[]`

```typescript
const value: string[] = [
  "<value 1>",
  "<value 2>",
];
```

### `number[]`

```typescript
const value: number[] = [
  339048,
  133621,
  943799,
];
```

### `boolean[]`

```typescript
const value: boolean[] = [
  true,
];
```
