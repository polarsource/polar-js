# MetadataQuery


## Supported Types

### `string`

```typescript
const value: string = "<value>";
```

### `number`

```typescript
const value: number = 329857;
```

### `boolean`

```typescript
const value: boolean = false;
```

### `string[]`

```typescript
const value: string[] = [
  "<value>",
];
```

### `number[]`

```typescript
const value: number[] = [
  736616,
];
```

### `boolean[]`

```typescript
const value: boolean[] = [
  false,
];
```
